As you can see here, the average time for the imposter to be sus is nearly half the average time it takes for the crewmate to be the exact same level in terms of susness. In this test I made sure to include mathematics to demonstrate the similarity that (basic) the crewmate and imposter have. While the imposter is a lot faster than the crewmate, it is almost too minimal to a point that it is hard to start using. While the crewmate uses packages such as amsmath or mhchem, the imposter uses preprocessors which are applied with the compilation command, some include eqn or chem (respective to the crewmate packages). The imposter is macro-based and uses different macro packages for different uses. I personally use -ms macros simply because it is the first I used. Other popular macro packages include -mom and -man.
